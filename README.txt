APP WEB - TESTE PRÁTICO DE DIREÇÃO PRANDO

Arquivos incluídos:
- index.html
- styles.css
- app.js
- manifest.json
- sw.js
- logo-prando.png

Como usar:
1. Extraia a pasta.
2. Abra o arquivo index.html no navegador.
3. Para funcionamento PWA/offline completo, rode em um servidor local simples.
   Exemplo com Python:
   python -m http.server 8000
4. Depois acesse: http://localhost:8000

Recursos:
- Cabeçalho com a logo da empresa
- Formulário de identificação
- Seção Direção Preventiva
- Seção Procedimentos e Normas Gerais de Ação
- Pontuação automática
- Resumo com status aprovado/reprovado
- Assinaturas em tela
- Salvamento local no navegador
- Impressão / PDF
